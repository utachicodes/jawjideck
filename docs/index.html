Features

One app for all your flight controllers

Everything you need to configure, tune, and fly - whether you're on ArduPilot, Betaflight, or iNav.

Real-Time Telemetry

Attitude indicator, GPS, battery, airspeed, altitude - all in a dockable, IDE-style dashboard you can customize.

Mission Planning

Interactive map with terrain-aware altitude profiles, spline curves, geofencing, and rally points. Upload/download via MAVLink.

PID Tuning & Configuration

Beginner-friendly presets, visual rate curves, servo mixer, modes wizard, and VTX settings for Betaflight, iNav, and ArduPilot.

OSD Simulator

See exactly what appears in your FPV goggles. Demo and live modes, 8 bundled fonts, PAL/NTSC support, 15+ OSD elements.

Parameter Management

All 800+ ArduPilot parameters with search, inline editing, real-time validation, modified tracking, and file import/export.

Firmware Flash

Multi-protocol board detection, STM32/DFU flashing, ArduPilot/Betaflight/iNav firmware with progress tracking. Supports F3 legacy boards.

CLI Terminal

Full xterm.js terminal with ANSI colors, tab autocomplete, command history, and parsed parameter suggestions from dump output.

Calibration Wizards

Step-by-step accelerometer and compass calibration with position diagrams, countdown timers, and arming flag-based status detection.

Legacy Board Support

Full GUI for F3-era boards (SPRacing F3, Naze32, CC3D). Same modern UI experience powered by CLI commands under the hood.
